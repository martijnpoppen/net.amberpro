Adds support for Amber Pro and Amber X

Retrieve information from your Amber. 
Get notification when a value has changed.

How to use:
- Install this app on your Homey.
- Go to new devices
- Insert password
- Device will be added
- Data will be fetched.
- Check statuses or create flows.

Current features:
- Reboot / switch off.
- Display CPU %
- Display Disk %
- Display Temperature
- Temperature alarm
- Display online time
- Upload files to Amber
- Amber One & Amber Plus: router functionality
    -- IP connected
    -- IP disconnected
- Get notification when a value has changed.
